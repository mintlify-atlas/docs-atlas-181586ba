Formbricks is a free and open-source surveying platform that helps you gather feedback at every point in the user journey. Build beautiful in-app, website, link, and email surveys with our no-code editor and leverage powerful analysis tools to turn responses into action.

## Why Formbricks?

Formbricks empowers teams to craft irresistible user experiences through continuous feedback. Whether you're validating a new feature, measuring product-market fit, or understanding user pain points, Formbricks provides the tools you need.

## Built on Modern Technologies

Formbricks is built with industry-leading open-source technologies:

- **Next.js** & **React** for a blazing-fast UI
- **TypeScript** for type safety and developer experience
- **Prisma** & **PostgreSQL** for robust data management
- **TailwindCSS** for beautiful, responsive design
- **Auth.js** for secure authentication
